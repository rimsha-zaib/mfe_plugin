contribplugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/myusername/tutor-contrib-contribplugin

Usage
-----

::

    tutor plugins enable contribplugin
